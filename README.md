# NavigationSecurityApp

NavigationSecurityApp is a simple Java Android Studio project that teaches Android navigation and component security.

## App Links

The example file `assetlinks_example.json` shows the Digital Asset Links content required for verified App Links.

This content must be hosted at:

```text
https://example.com/.well-known/assetlinks.json
```

Replace `SHA256_FINGERPRINT_OF_YOUR_APP_SIGNING_CERTIFICATE` with the real SHA-256 fingerprint of the app signing certificate.

## Important

Do not execute Gradle commands for this setup task. The project files are intended to be opened in Android Studio.
